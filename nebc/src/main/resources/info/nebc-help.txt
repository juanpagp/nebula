OVERVIEW: Compiler for the Nebula language.
USAGE: nebc [options] file_or_dir...
OPTIONS:
  -h, --help                Show this help message and exit.
  -v, --version             Show compiler version and exit.
  --verbose             Enable verbose debug logging.
  -e, --entry <name>        Specify the entry point for an executable.
  -o, --output <file>       Specify the output file name.
  -L <path>                 Add a directory to the library search path.
  -l <name>                 Link a specific library (e.g., 'std' for libstd.so).
  -s, --symbols <name>      Load symbols from a .nebsym file (e.g., 'std').
  -B, --library             Compile as a library (.nebsym + .a/.so).
  -t, --target <platform>   Specify the target platform (e.g., 'linux-x64').
  -k, --check               Run semantic analysis only; do not generate output.
  -n, --native <files...>   Explicitly include native source files (.cpp, .h) in the build.
  -S, --static              Link for a static, freestanding binary (no libc).
  --nostdlib                Do not load the standard library automatically.
FLAGS:
  --borrow-checking=<level> Set borrow checking level: none, allowed (default), enforced.
  --ignore-warnings         Compile while ignoring all warnings.
